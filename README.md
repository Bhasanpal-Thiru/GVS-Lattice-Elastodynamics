## Overview
This repository contains the computational frameworks, theoretical derivations, and numerical proofs for **Geometric Vacuum Scaling (GVS)**.

This repository provides the data and code necessary to reproduce the paper's claims, demonstrating that both macroscopic gravitational anomalies (Dark Matter) and quantum topological stability can be unified under a single lattice elasticity framework.
